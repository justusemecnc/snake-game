# Snake Game

A simple Python and Pygame snake game with menu, gameplay, pause, and game-over scenes.

## What it does

- Classic snake gameplay with arrow keys or WASD
- Three difficulty levels: Easy, Medium, Hard
- Score tracking and persistent high score
- Pause/resume support
- Settings saved to disk

## Requirements

- Python 3.10+
- Pygame

## Run locally

1. Install the dependency:

   ```bash
   pip install pygame
   ```

2. Start the game:

   ```bash
   python main.py
   ```
